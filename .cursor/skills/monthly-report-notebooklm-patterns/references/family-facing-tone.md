# ご家庭向け月次レポート：文体・プロンプト指針

NotebookLM の `notebook_query` や、Markdown／HTML への清書時に **第三者の観察レポートのようにならない** ための指針です。

## 避ける表現（他人事・過度に客観）

- 「〜が挙がっております」「〜と記載されています」「〜と整理です」「推奨されております」だけの**観察者口調**の連続。
- 「〜に見えます」「〜段階に見えます」など、**当塾の関与が消える**言い方。
- 生徒名・担当名を出さず「一般論」だけを並べる文。
- 「順次ご説明してまいります」など、**誰が・いつ・何を**が抜けた抽象句の多用。

## 推奨する立ち位置

- **当塾／担当教師の一人称**（「当塾では」「森田としては」「私ども」）と、**お子さまのお名前（○○さん）**を適度に織り込む。
- 事実はマップ許可ソースに限定しつつ、**見立て・ねらい・お願い**は「私たちが伴走している」文脈で書く。
- 課題は「データ上の欠陥」ではなく、**一緒に力を入れたい点**として伝える。
- 保護者への依頼は「ご協力をお願いします」より、**「こうしていただけると、授業がさらに活きます」**のように理由とセットにする。

## NotebookLM `notebook_query` に足す一文（例）

クエリの末尾に次を**必ず含める**こと。

> 出力は保護者向けの月次レポート用です。観察者や第三者のレポート調（「〜と記載があります」「〜が挙がっております」だけの連続）にせず、当塾・担当教師がお子さまに伴走している口調（です・ます）で書いてください。生徒のお名前を適度に入れ、推奨や課題は「当塾では〜したい」「森田として気にかけている点は〜」のように当事者として述べてください。ソースにないことは書かない。**03 授業内容**では、表の直下に「理解度（★）の目安」を次の5行を**一字一句変えず**含めること：★☆☆＝基礎内容の定着が必要／★★☆＝基礎問題が解けるが応用問題に対応できない／★★★＝基礎に加え応用問題にも対応できている／※いずれも授業内の観察と演習の様子に基づきます。別の言い回しに置き換えない。

## 03 授業内容 — 理解度（★）の目安（固定文案 HTML／MD 共通）

**授業日・単元の表の直下**に、次を **そのまま** 載せる（意訳・略・別定義に差し替え禁止）。

| 記号 | 意味 |
|------|------|
| ★☆☆ | 基礎内容の定着が必要 |
| ★★☆ | 基礎問題が解けるが応用問題に対応できない |
| ★★★ | 基礎に加え応用問題にも対応できている |

※いずれも授業内の観察と演習の様子に基づきます。

学習計画表に「理解度」列がない場合は、その旨を **同じブロックの末尾** に一文足す（例：「学習計画表に理解度列がないため、担当の見立てで記載しています。」）。正本は **`docs/samples/monthly-reports/monthly_pattern_b_content.template.md`** の **03 授業内容** 節。

## 推敲チェック（第2回・第3回に追加してよい観点）

- [ ] 「記載があります／整理です」が残っていないか → **当事者の文に言い換えたか**
- [ ] 生徒名・担当名が十分に入っているか
- [ ] 保護者が読んだときに**冷たい調査報告**に聞こえないか
- [ ] **03（授業内容）** の表の直後に、**理解度（★）の目安**が固定文案どおり入っているか（言い換え・欠落がないか）

## 書き換え例（同一事実）

| 避けたい | 望ましい方向 |
|----------|----------------|
| 春休みに向けての課題として、トピック A の発展問題への慣れが挙がっております。基本の定着に加え、ExamMate など…推奨されております。 | 春休みに向けて、**当塾では**トピック A の発展問題に**少しずつ慣れていくこと**を大切にしたいと考えております。謙吾さんには、基本の定着に加え、ExamMate など試験に近い形式の問題にも触れていただき… |
| 計画どおり進んでいる段階に見えます。トピック C に入る前後は、今後の授業とあわせて順次ご説明してまいります。 | 学習計画表では、トピック A・B の多くで学校・当塾とも「済」が入っており、**謙吾さんのペースは計画とよく合っている**ように思います。トピック C に入るころは、**森田の授業の中で**理解の度合いに合わせて、改めて丁寧にご説明いたします。 |
