# 月次案 × 月次マップ × Pattern B 対応表（Step 1）

**根拠スプレッドシート**: 【EB塾】月次レポート案（Spreadsheet ID は運用の URL から取得）  
**参照タブ**: `イメージ`（gid `1063169303`）— 一次データは [data/EB_monthly_report_gid1063169303_values.json](../../../data/EB_monthly_report_gid1063169303_values.json) に gws 取得例あり。

**凡例**: 「月次マップ（データソース）」列は、当該タブ右端付近のセル注記（サンプルイメージ）を転記したもの。実運用ではスプレの**正式な月次マップ行**に差し替えること。

## MD 見出し ↔ HTML レイアウトひな型（`monthly_pattern_b_template.html`）

月次案の ③ は「塾での様子」と「授業内容」の 2 本。**配布 HTML の `span.sec-num` と見出し語**は **`02` と `03` で対応**させる（**`03a` / `03b` は使わない**）。KPI はマップに行が無ければ HTML から `.kpi-row` を削除する。

| MD（テンプレ） | HTML テンプレ内のセクション | Pattern B 系クラス |
|----------------|-----------------------------|-------------------|
| 01 基本情報 | 01 | `.info-table` |
| KPI（任意） | （任意ブロック） | `.kpi-row` |
| 02 塾での様子 | 02 | `.section-heading` + mood 表 + `.analysis-box` |
| 03 授業内容 | 03 | `.data-table` + `.analysis-box` |
| 04 課題とアドバイス | 04 | `.analysis-box` |
| 05 学習の進捗 | 05 | `.data-table`（＋任意 `.bar-gauge`） |
| 07 今後の授業計画 | 07 | `.plan-table` + `.analysis-box` |

**学校の試験**: 原則 **01 に集約**し、独立セクションを置かない（[monthly_pattern_b_content.template.md](monthly_pattern_b_content.template.md) 参照）。

**参照**: 完成イメージ・余分なセクション例（面談候補など）は `src/reports/templates/pattern_b.html`。月次案に無いブロックはテンプレに載せない。

## 大ブロック対応

| 月次案ブロック | MD 見出し（推奨） | 月次マップ（データソース）注記 | Pattern B コンポーネント |
|----------------|-------------------|----------------------------------|---------------------------|
| ①基本情報 | `## 01 基本情報` | 学習計画表_student sheet | `.info-table` |
| ②学校の試験 | （省略可）`## 学校の試験（省略可）` または 01 に集約 | 学習計画表_student sheet（日程は※指導モニタリングのスプレ、等の注記あり） | `.data-table` + 注釈 |
