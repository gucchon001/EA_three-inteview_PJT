2. メールの構成とテンプレ

基本構成：挨拶と目的→学習進捗と伸びている点（数値で肯定）→現状の課題と目標（逆算で〇回不足）→解決策とA案・B案・C案（料金明記）→ご協力・三者面談案内。件名：【EDUBAL Academy】〇〇様の学習進捗および今後の学習計画について。本文は進捗と良い点を数値で、課題を逆算で、提案をA案（現状＋宿題）B案（時間延長or週2回・料金）C案（長期休暇集中）で選択肢提示。フレーズ集：肯定的報告・逆算スケジュール・演習不足・長期休暇の4パターンで納得しやすい理由を記載。

3. ご家庭別分析
