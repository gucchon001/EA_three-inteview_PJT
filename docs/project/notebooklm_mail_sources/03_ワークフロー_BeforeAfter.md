週次ルーチン（P3：週次指導報告の自動化）

Before

指導（授業）
  └→ 教師が手動で指導報告書を提出（フォーム等）
       └→ 担当者が月末にダウンロード → FileMakerへ手動取り込み
            └→ 請求書作成（月末集中）


項目

現状

報告書作成

教師が手動・フォーマット不統一

データ取り込み

保護者への共有

遅延しやすい・未共有になることあり

After

指導（授業）
  └→ 文字起こし（自動）
       └→ 報告書 自動生成（1w サイクル）
            └→ 保護者へ自動送付（承認プロセスなし）
                 └→ 請求書が事実上の承認


項目

--------------------------------------------------------------------------------


データ収集（P4：無料見積もり診断サービス）

Before

答案・模試結果の回収を口頭で依頼
  └→ 回収率 0〜1割（動機付けなし）
       └→ プランニング精度低い → 提案が刺さらない


After

無料見積もり診断サービスを入口に設定
  └→ 「情報を出すと分析レポートがもらえる」インセンティブ設計
       └→ 答案・模試・スケジュール提出 → AI分析レポート（PDF）提供
            └→ 回収率向上 → プランニング精度向上 → アップセル提案精度向上
