小林 慧さま 面談・教師MTG 時系列まとめ

小林 慧さまの「ご家庭に提案」および「中野教師MTG」の記録をもとにした、学習進捗・課題・指導方針の時系列推移です。

1. 2025年9月〜10月

状況・進捗

: 学校の授業が実験中心で理論的な解説が少なく、知識が断片的になっていることが懸念されました。

課題

: 基礎がまだ固まっていないため、まずは基礎を確実に固めることが大切だと伝えられました。また、現在のペースでは3月のMockに間に合わないことが課題として挙げられました。A案（週1回レクチャー中心）で進めることになりましたが、宿題が滞りがちであり、自己採点では正答率8割と申告されているものの、レクチャー中心の授業では本当の理解度を測りきれていないという不安要素が共有されました。

提案・指導方針

: 授業をスムーズに進めるため、「予習ノート」を用いた授業スタイルが有効であると確認されました。当面（あと1ヶ月程度）はA案のままレクチャー優先で教科書を進め、宿題で復習と理解度測定を行う方針を継続することになりました。テスト対策として、予習・復習内容を「自分だけのまとめノート」として手元に残す学習アドバイスを提案しました。

2. 2025年11月〜12月

状況・進捗

: 学習ペースは順調で1月までに全単元終了が見込めるものの、授業内容を定着させるための宿題の提出が十分でないことが最大の課題として報告されました。

課題

: 宿題が一切提出されていない状況が続いており、このままでは目標スコア（5〜6）の達成は難しいという強い懸念が共有されました。また、メートルをキロメートルに直さないなどの「ケアレスミス」も課題として挙がりました。

提案・指導方針

: 根本的な原因究明のため12月に三者面談を行うことになりました。また、宿題提出の強制力を高めるため、今後はユニットテストを「テイクホームテスト」形式にして教師が採点し、結果を保護者に送付して理解度と提出状況を可視化する方針が提案されました。

3. 2026年1月

状況・進捗

: 宿題は遅れつつも提出されるようになりました。しかし、学校で未習の範囲を丁寧に指導していることに加え、生徒が「予習ノート」の事前記入を怠っているため、授業の効率が落ち、全体の進捗が遅れ気味になっていることが確認されました。

課題

: 未習部分の補足と予習不足により進行に時間を要し、Mock試験までに内容が間に合わない（5回分不足）ことが伝えられました。宿題の提出状況は改善し基礎力は向上しましたが、Mockに向けて応用問題や記述式問題への対応力強化が課題となりました。

提案・指導方針

: 遅れを取り戻すため、1回の授業で2つのサブトピックを進めるとともに、不足する約3回分の授業について、「補填回数を増やす」「授業時間を延長する」「宿題で対応する」の3つの選択肢をご家庭に提案することになりました。また、記述式問題での説明不足による失点を防ぐ指導を強化します。

4. 2026年2月

状況・進捗

: 前回のテスト（トピックE）でスコア6を取得して自信をつけています。宿題と予習ノートがコンスタントに提出されるようになり授業はスムーズに進んでいます。

課題

: 発展問題になると解き方がわからなくなることや、頭の中で計算して途中式を省略してしまうことで部分点を取り切れていないことが新たな課題として報告されました。

提案・指導方針

: 今後は演習を通して、途中式をしっかり書く習慣を身につけられるよう指導していく方針が伝えられました。2月15日までに全単元の学習を一通り終え、Mockテストの結果を確認後、3月からは過去問演習に入るスケジュールと、2月の三者面談のご案内が提示されました。
